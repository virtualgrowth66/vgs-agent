Speech-to-Text

Real-time speech recognition using the Web Speech API. Visitors speak naturally, the engine transcribes instantly. Supports 10+ languages.

Text-to-Speech

Natural-sounding voice responses using browser speech synthesis. Auto-detects the best voice for each language. Speak button on every bot message.

Booking Flow

Full conversation flow: greet → ask → answer → detect booking intent → collect name/contact/date/time/service → confirm. Progress bar shows the step.

Intent Detection

Smart pattern-matching engine identifies greetings, pricing questions, hours inquiries, booking requests, contact info, language changes, and more.

Multi-Language

Auto-detects visitor language from their speech or text. Responds in the same language. Supports English, Spanish, French, German, Italian, Portuguese, and more.

Pluggable LLM

Rule-based engine works out of the box with zero API keys. Clear integration point to swap in OpenAI, Anthropic, or any LLM for AI-powered responses.

Embed Snippet

Add these two scripts to your website's </body> tag area: 

<!-- SiteMind AI — Voice Engine + Widget -->
<script src="voice-engine.js"></script>
<script src="widget.js"
        data-agent-id="demo-agent"
        data-language="en"
        data-primary-color="#6366F1"
        data-auto-detect="true"
        data-voice="true"
        data-tts="true"></script>
<!-- SiteMind AI — With Backend Server -->
<script src="voice-engine.js"></script>
<script src="widget.js"
        data-agent-id="demo-agent"
        data-primary-color="#6366F1"
        data-server-url="https://your-server.com"
        data-voice="true"
        data-tts="true"></script>
<!-- SiteMind AI — Single File Bundle -->
<script src="embed.js"
        data-agent-id="demo-agent"
        data-language="en"
        data-primary-color="#6366F1"></script>
